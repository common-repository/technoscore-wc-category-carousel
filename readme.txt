== Description ==

Technoscore WC Category Carousel is  best plugin for WordPress to showcase Woocommerce products categories images in a nice sliding manner. 

= More details are as follows:- =
* It shows Woocommerce products categories images in Carousel.
* It adds a 'Add WC Product Category Carousel' tag in Post Content Editor of post/page to insert shortcode for WC Product Category Carousel in post content area to show on frontend of the particular post/page.
* It provides options to add custom css class in shortcode to change the look and feel of WC Product Category Carousel .
* It provides options to change the navigation, loop, number of images to show at full width screen setting from admin panel. 
* WC Product Category Carousel is responsive in nature by default.
* Creates shortcode for WC Product Category Carousel
	Example: [techno_wc_list_init  carousel_class ='add-class-name'  ids='add-wc-cat-ids']

== Installation ==

1. Use WordPress' built-in installer
2. Download the ZIP file from WordPress website
3. Extract the technoscore-wc-category-carousel directory to your computer
4. Upload the technoscore-wc-category-carousel directory to the /wp-content/plugins/ directory
5. Activate the plugin in the Plugin dashboard
